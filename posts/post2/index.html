Post 2: Ultrasonic Sensor with Arduino

Published by TheTechGuy | Dev Circuit

The ultrasonic sensor is one of the most useful components in beginner and intermediate Arduino projects. It allows you to measure distance using sound waves.

🔹 What is an Ultrasonic Sensor?

The most common model is the HC-SR04. It sends out ultrasonic sound waves and measures the time it takes for the echo to return.

Using that time, we calculate distance.

🔹 Pins of HC-SR04

The sensor has 4 pins:

  • VCC – 5V power
  • Trig – Trigger pin (send signal)
  • Echo – Receive reflected signal
  • GND – Ground

🔹 Basic Wiring

Example connection with Arduino UNO: 

VCC  → 5V
GND  → GND
Trig → Pin 9
Echo → Pin 10

🔹 Basic Arduino Code

const int trigPin = 9;
const int echoPin = 10;

long duration;
int distance;

void setup() {
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  Serial.begin(9600);
}

void loop() {

  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);

  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);

  duration = pulseIn(echoPin, HIGH);

  distance = duration * 0.034 / 2;

  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.println(" cm");

  delay(500);
}

🔹 Important Things To Know

• Accurate range is about 2cm to 400cm • Works best on flat surfaces • Soft materials absorb sound • Avoid placing objects at sharp angles

🔹 Common Beginner Mistakes

• Forgetting to divide by 2 • Not adding delay • Wrong wiring of Echo/Trig • Not using proper power supply

🔹 Project Ideas

• Distance meter • Obstacle avoiding robot • Smart parking system • Water level detection
