Ultrasonic Sensor with Arduino

Published by TheTechGuy | Dev Circuit

The ultrasonic sensor is one of the most commonly used distance measuring components in Arduino projects. It works using sound waves instead of light.

What is the HC-SR04?

The HC-SR04 ultrasonic sensor sends out high-frequency sound waves and listens for the echo. By measuring the time it takes for the sound to return, we can calculate the distance.

Sensor Pins

The module has 4 pins:

• VCC → 5V • GND → Ground • Trig → Trigger signal • Echo → Receives reflected signal

Basic Wiring

VCC  → 5V
GND  → GND
Trig → Pin 9
Echo → Pin 10

Basic Arduino Code

const int trigPin = 9;
const int echoPin = 10;

long duration;
int distance;

void setup() {
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  Serial.begin(9600);
}

void loop() {

  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);

  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);

  duration = pulseIn(echoPin, HIGH);

  distance = duration * 0.034 / 2;

  Serial.print("Distance: ");
  Serial.print(distance);
  Serial.println(" cm");

  delay(500);
}

Important Things to Know

• Accurate range is between 2cm and 400cm • Works best on flat solid surfaces • Soft materials absorb sound • Always divide by 2 in the distance formula
